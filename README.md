This is to be a fully functioning part of the AMLS system from the side of the platform. 

Controller should be operating with such data as its own GPS location, illumination around it, its speed and the data that would be sent from the drone which is descripted in _Data packet structure paragraph_ in Liberty-Way Projecta
So far, this system is able to measure the level of illumination, the speed of the platform, and also turn on/off the backlight
